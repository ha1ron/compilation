Реализация алгоритма Дейкстры для нахождекния кратчайшего пути в звешенном графе.
Реализация вынесена в функциональный модуль, ребра считываются из таблицы в БД, для ускорения работы, разбитие логики на подпрограммы было убрано.
Структуры таблицы с ребрами и с результирующей структурой находятся в output-structure.abap
